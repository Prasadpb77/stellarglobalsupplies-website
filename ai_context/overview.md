# Overview

## Project Name
Stellar Global Supplies Web Platform

## One-Liner
Production-grade B2B industrial supply platform for SS/MS materials and fasteners with 500+ products, featuring AI-powered blog automation.

## Problem It Solves
Industrial buyers, manufacturers, and fabricators struggle to find reliable suppliers for Stainless Steel, Mild Steel, and Fastening products in India. The market is fragmented with inconsistent quality, unclear pricing, and poor digital presence from traditional suppliers. This creates procurement friction, quality risks, and time wasted on vendor verification.

## Solution
A modern, SEO-optimized web platform that serves as a digital catalog and enquiry portal for Stellar Global Supplies. The platform showcases 500+ products across 3 categories with detailed specifications, enables direct contact for quotes, and establishes online credibility through professional design, fast performance, and comprehensive technical information. Built with Next.js for optimal Core Web Vitals and deployed on AWS for global reach. Features an AI-powered blog automation system using AWS Bedrock Nova Pro that automatically generates SEO-optimized blog pages from markdown files.

## Who Uses It
- B2B buyers at manufacturing companies (automotive, aerospace, construction)
- Fabricators and machine shops needing SS/MS raw materials
- Procurement managers at engineering firms
- Industrial contractors sourcing fasteners and fittings
- Small to medium enterprises requiring reliable steel suppliers in Pune, Maharashtra

## Status
Stable / Actively maintained

## Blog Automation
The platform includes an AI-powered blog system that automatically implements blog functionality when new markdown files are added to content/blog/. Uses AWS Bedrock Nova Pro to generate SEO-optimized, marketing-focused blog pages with zero manual coding required.
